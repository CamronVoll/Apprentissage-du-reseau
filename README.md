## Introduction aux Réseaux

Un **réseau informatique** est un ensemble d'appareils interconnectés permettant l'échange de données.  
Il peut être local (LAN) ou étendu (WAN) et peut utiliser différents protocoles et technologies pour fonctionner.

---

## Protocoles et Modèles

- **TCP/IP** : ensemble de protocoles pour la communication sur Internet.  
- **Modèle OSI** : modèle en 7 couches pour structurer la communication réseau.  
- **Noeud DTE / DCE** : dispositifs terminaux et équipements de communication.  

---

## Trafic et Sécurité Réseau

- **Trafic réseau** : données circulant entre les appareils d’un réseau.  
- **Vulnérabilité** : faille permettant une attaque sur le réseau.  
- **Malwares et Phishing** : menaces informatiques courantes à connaître.  

---

## Adresses IP et Routage

- Une **adresse IP** identifie chaque appareil sur un réseau.  
- Le **routage** permet aux données de circuler d’un réseau à un autre efficacement.
